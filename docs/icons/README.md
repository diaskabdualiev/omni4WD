# PWA Icons

This directory should contain PWA icons for the web app.

## Required icons:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## How to create:

1. Create a simple robot icon or use an online generator
2. Export as PNG at the required sizes
3. Place files in this directory

## Quick generation:

You can use online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or create a simple colored square with a robot emoji as a placeholder.
